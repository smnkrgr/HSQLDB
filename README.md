2. Download HSQLDB [hier](http://hsqldb.org/)
3. Einbinden der hsqldb-2.4.0/hsqldb/lib/hsqldb.jar in das Eclipse Projekt über Project > Properties > Java Build Path: Reiter "Libraries" > "Add external JARs"
